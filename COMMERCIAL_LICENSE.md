## 中文

### 概述

AreaMatrix 默认以 [PolyForm Noncommercial 1.0.0](LICENSE) 协议发布，**禁止商业使用**。本文档说明如何获取商业授权。

### 哪些情况需要商业授权

如果你或你所在的组织希望以下列任一方式使用 AreaMatrix（或基于其源码的衍生作品），**必须**先获取商业授权：

- 销售 AreaMatrix 或包含 AreaMatrix 代码的软件产品
- 在 SaaS / 云服务中托管 AreaMatrix 并向用户收费
- 将 AreaMatrix 代码嵌入商业产品并对外销售
- 为客户提供基于 AreaMatrix 的有偿定制开发服务
- 其他任何能产生直接或间接商业收入的使用方式

### 哪些情况不需要商业授权

以下用途由 PolyForm-NC 默认许可证覆盖，**无需**额外授权：

- 个人使用、学习、研究
- 学术机构、非营利组织内部使用
- 企业内部业务运营（不对外销售）
- Hobby 项目、爱好者社区使用
- 提交 PR、Issue、文档贡献

### 如何申请商业授权

请通过以下任一方式联系：

- Email: `commercial@<your-domain>` *（占位，请部署时替换为实际邮箱）*
- GitHub: 在主仓库创建标题为 `[Commercial License Inquiry]` 的 issue

申请时请提供：

1. 你的姓名 / 公司名称
2. 预期使用场景与规模（用户数、部署方式、是否对外销售）
3. 期望的授权类型（永久 / 订阅）
4. 是否需要修改源码权利
5. 联系方式

### 授权流程

```mermaid
flowchart LR
    Inquiry[提交申请] --> Review[需求评审 1-3 工作日]
    Review --> Quote[报价 + 草拟协议]
    Quote --> Negotiate[条款协商]
    Negotiate --> Sign[签署协议]
    Sign --> Pay[支付授权费]
    Pay --> Granted[授权生效]
```

### 授权类型与定价

具体定价根据使用规模一事一议。常见档位（仅供参考）：

| 档位 | 适用场景 | 价格区间 |
|---|---|---|
| Startup | 年营收 < 100 万 / 用户数 < 1000 | 协商 |
| SMB | 中小企业 | 协商 |
| Enterprise | 大型企业 / SaaS 部署 | 协商 |
| OEM | 嵌入到自有产品对外销售 | 协商 |

### 授权协议涵盖

- 商业使用权（覆盖 PolyForm-NC 限制的部分）
- 技术支持（视档位而定）
- 不再受 PolyForm-NC 的"必须保留 copyright"限制（视协议条款）

## English

### Overview

AreaMatrix is distributed under the [PolyForm Noncommercial 1.0.0](LICENSE) license by default, which **prohibits commercial use**. This document describes how to obtain a commercial license.

### When You Need a Commercial License

You or your organization **must** obtain a commercial license to use AreaMatrix (or derivatives of its source code) in any of the following ways:

- Selling AreaMatrix or software products that include AreaMatrix code
- Hosting AreaMatrix in a SaaS / cloud service and charging users
- Embedding AreaMatrix code into a commercial product for sale
- Providing paid custom development services based on AreaMatrix
- Any other use that generates direct or indirect commercial revenue

### When You Don't Need a Commercial License

The default PolyForm-NC license already covers:

- Personal use, learning, research
- Academic institutions, nonprofit internal use
- Internal business operations (not resold)
- Hobby projects, hobbyist community use
- Submitting PRs, issues, documentation contributions

### How to Apply

Contact us via either:

- Email: `commercial@<your-domain>` *(placeholder; replace with actual address upon deployment)*
- GitHub: Open an issue titled `[Commercial License Inquiry]` in the main repository

Please include:

1. Your name / company name
2. Intended use case and scale (user count, deployment model, resale plans)
3. License type desired (perpetual / subscription)
4. Whether source modification rights are needed
5. Contact information

```mermaid

```
